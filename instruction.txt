Do this after pulling from GitHub:

1. Open Terminal run this command:
    npm create vite@latest myportfolio-react -- --template react 
        // This installs React JS plus Vite 
    if your using typeScript use this command
    npm create vite@latest myportfolio-react -- --template react-ts
2. Then move to your directory by doing this in the terminal:
    cd myportfolio-react    
        //make sure your are using the same file name
3. Install node modules use this command to your terminal
    npm install
4. Run your React Project by using this command
    npm run dev
5. Install bootstrap dependencies using this command 
    npm install bootstrap
6. Open src/main.jsx and add the Bootstrap CSS import at the top:
    This let you access bootstrap globally

import React from "react";
import ReactDOM from "react-dom/client";
import App from "./App.jsx";
import "./index.css";

// Bootstrap CSS 
import "bootstrap/dist/css/bootstrap.min.css";

// Bootstrap JS (optional, for things like modals, tooltips)
import "bootstrap/dist/js/bootstrap.bundle.min.js";

ReactDOM.createRoot(document.getElementById("root")).render(
  <React.StrictMode>
    <App />
  </React.StrictMode>
);
